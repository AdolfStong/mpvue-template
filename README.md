## 安装
步骤 ① **npm install** ② **npm run dev**

## 打包
测试： **npm run test**
正式： **npm run build**

## 目录结构

src/main.js  入口文件

api --- 接口路径配置

components --- 组件库

config --- 域名路径

contants --- 0SS img 地址

css --- 样式库

mixins --- vue 混入

pages --- 前端页面及路径

router --- 前端路由

store --- vuex 存储库

utils --- 共享方法、插件


## 技术栈
* webpack (3.11.0)
* Node (8.11.2)
* mpvue (1.0.11)
* ES6
* 微信小程序
